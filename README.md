# bluebankProj
É uma apilicação sigle-page application integrando um front-end angularjs com um back-end node.js de
uma lista telefonica

#Requisitos
-Para testar, é necessário ter o node.js instalado na máquina.
-Primeiro é necessário executar o arquivo server.js que está na past Node do projeto. Na linha de comando, vá
no  caminho da pasta Node e digite o comando node server.js
Logo em seguida abra a aplocação no brower.
  Para abrir a versão de dev, clique duas vezes no arquivo ListaTelefonica.html que está no projeto
  Para abrir a versão que está mimetizado através do gulp, vá na pasta dist e clique duas vezes no arquivo
  ListaTelefonica.html que está lá.
  
  #Funcionalidades
  
  #Tela Inicial
  -Na primeira tela, serão exibidos os contatos que estão na base, é possível deletar um contato(ou mais de um ao mesmo tempo)
  clicando na checkbox correspondente ao contato, e em seguida no botão "Apagar Contatos", que surgirá enquanto algum contato estiver
  selecionado.
  -Para saber detalhes e informações correspondentes, basta apenas clicar sobre o nome do contato que o link direcionará para 
  a página de detalhes. 
  -Nessa tela é possível procurar um contato digitando no campo "O que você está buscando?"
  -Para inserir um contato, clique no botão "Novo Contato"
  
  #Tela de inserção
 -Existirão campos a serem preenchidos com os dados do contato
 -O botão "Adicionar Contato" só será abilitado se os requisitos para preenchimento dos dados forem atendidos
 -Apos isso clique no botão "Adicionar Contato" para que ele seja adicionado a lista
